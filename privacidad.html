Política de privacidad

Responsable

Responsable del servicio: Alberto Ambroj.
Email de contacto: aambroj@yahoo.es

Datos que pueden recibirse

Esta web no usa formularios con base de datos propia. Cuando una persona contacta por email, puede enviar voluntariamente nombre, email, URL de su producto, descripción del proyecto, métricas, capturas, documentación u otra información necesaria para solicitar un diagnóstico.

Finalidad

Los datos se utilizan para responder a solicitudes, preparar diagnósticos de producto, mantener comunicaciones relacionadas con el servicio y entregar informes contratados.

Base de tratamiento

La base es el consentimiento del usuario al escribir voluntariamente y, si se contrata el servicio, la ejecución de la relación solicitada.

Conservación

La información se conservará durante el tiempo necesario para responder, prestar el servicio y atender posibles comunicaciones posteriores. El usuario puede solicitar la eliminación escribiendo al email de contacto.

Cesión de datos

No se venden datos personales. No se ceden a terceros salvo obligación legal o necesidad técnica imprescindible para prestar el servicio.

Derechos

El usuario puede solicitar acceso, rectificación, eliminación u oposición escribiendo a aambroj@yahoo.es.

Confidencialidad de proyectos

La información enviada sobre productos, ideas, métricas o estrategias será tratada con discreción. No se publicarán casos reales sin autorización expresa.

Nota: esta política es una versión inicial para una web estática. Antes de operar como actividad comercial definitiva, conviene revisarla con asesoramiento legal y fiscal.
