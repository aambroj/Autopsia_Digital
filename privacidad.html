Política de privacidad

Esta web recoge únicamente los datos que el usuario decide enviar voluntariamente por correo electrónico al solicitar información o contratar un diagnóstico.

Responsable

Responsable del servicio: Alberto Ambroj.

Datos tratados

Podemos recibir nombre, email, URL del producto, descripción del proyecto y cualquier información que el usuario incluya voluntariamente en su mensaje.

Finalidad

Los datos se utilizan exclusivamente para responder a la solicitud, preparar el diagnóstico y mantener la comunicación relacionada con el servicio.

Conservación

Los datos se conservarán solo durante el tiempo necesario para prestar el servicio y atender posibles comunicaciones posteriores.

Cesión de datos

No se venden datos personales ni se ceden a terceros salvo obligación legal.

Derechos

El usuario puede solicitar acceso, rectificación o eliminación de sus datos escribiendo al email de contacto.

Nota: antes de publicar esta web como actividad comercial definitiva, conviene completar los datos legales obligatorios según la situación fiscal real del titular.
