# Caso demo ficticio - HabitPro

## Objetivo

Mostrar un ejemplo de Autopsia Digital sin usar datos reales de clientes.

## Producto ficticio

- Nombre: HabitPro
- Tipo: app de hábitos y productividad personal
- Problema: muchas descargas, pocas conversiones a plan premium

## Diagnóstico

HabitPro no falla por tecnología. Falla porque su promesa comercial es demasiado genérica.
Parece otra app de hábitos más y no ofrece una razón clara para pagar.
