# Flujo de cobro - Autopsia Digital

## Situación actual

La web no tiene pasarela de pago automática.
El contacto y la confirmación se hacen por email.

## Flujo recomendado

1. El cliente escribe desde la web.
2. Se revisa si el producto encaja.
3. Se recomienda plan: Express, Completa o Premium + Mercado.
4. Se confirma alcance, precio y plazo por email.
5. Se acuerda forma de pago.
6. Se empieza el análisis.
7. Se entrega el informe.
8. Se solicita feedback.

## Importante

No pedir ni aceptar pagos sin confirmar antes por email:

- Producto analizado.
- Plan contratado.
- Precio.
- Plazo.
- Formato de entrega.
- Condiciones básicas.

## Futuras opciones posibles

- Stripe Payment Links.
- PayPal.
- Gumroad.
- Lemon Squeezy.
- Ko-fi.
- Transferencia bancaria.
- Facturación manual.

## Recomendación inicial

Para los primeros clientes, mantener pago manual y controlado.
Cuando haya validación real, elegir una pasarela sencilla.
